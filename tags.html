Tags basicas de formatação de texto:

h1 = cabeçalho e tem tamanhos até "6" // ALÉM DISSO O H1 É UTILIZADO COMO MOTOR DE BUSCA DO GOOGLE, E INFLUENCIA NO SITE, E COMO ELE VAI SER ENCONTRADO.

strong = deixa o texto em negrito

p = parágrafo, ele dita onde aconteceram espaçamentos e parágrafos!

u = deixa o texto sublinhado

hr = linha horizontal

OBS: sobre a formatação de textos em um site, com uso das tags citadas, é possivel usar as variações de todas em uma linha de código!

br= é utilizado para quebra de linha, pode ser usado repetidas vezes para aumentar o espaçamento.

ul; li= é uma lista desornedana, o que representa a lista desornedana é o "ul" , ja o "li" representa os itens dessa lista. Por ser uma lista desordenada , óbviamente, a sequência de itens não representa a ordem que devem ser utilizados. Como no exemplo abaixo:

  • amor
  • carinho
  • paixão

ol; li= Já aqui, é uma lista ordenada, no caso usarei os itens que são sequênciais, em uma relação utilizando os itens que foram listados em desordem anteriormente para melhorar a lógica:

  1. paixão
  2. carinho
  3. amor
